# IwaraTV 用户手册

简体中文 | [English](user-manual.en.md)

IwaraTV 是一个用于浏览 Iwara.tv、打开视频详情、启动本地播放器和下载视频的桌面工具。它在本地运行；Iwara 账号访问、Cloudflare 验证、播放器路径、播放历史和下载记录都保留在你的机器上。

本项目为独立制作，不隶属于 Iwara，也未获得 Iwara 的赞助、认可或背书。

## 安装与启动

1. 从 [GitHub Releases](https://github.com/demogest/IwaraTV/releases) 下载最新版本。
2. 安装对应平台的包；如果有 Windows 便携版，也可以直接使用便携构建。
3. 启动 IwaraTV。
4. 首次播放前先打开 **设置**，检查 MPV 或外部播放器状态。

Windows 发布构建可以携带内置 MPV。macOS 和 Linux 用户需要自行安装 MPV，或在 **设置** 中配置其他外部播放器。

## 主导航

- **浏览**：最新、当前人气、长期热门、关注标签和指定作者视频列表。
- **搜索**：按关键词搜索，并可按相关、最新、播放或喜欢排序。
- **订阅**：登录或完成会话验证后查看订阅作者的视频。
- **收藏**：本地保存的视频，并提供备份、导出和导入工具。
- **下载**：查看下载队列、进度、重试、打开文件/文件夹和下载历史。
- **历史**：本地保存的最近播放记录。
- **设置**：账号/会话状态、播放器路径、下载默认值、媒体线路、X-Version 和标签偏好。

## 打开视频

可以通过以下方式打开视频：

- 点击视频列表或搜索结果中的视频卡片。
- 在顶部输入 Iwara 视频 ID。
- 在顶部粘贴 Iwara 视频 URL。
- 从视频详情进入作者页，浏览该作者的视频。

详情页会列出可用清晰度。选择清晰度后，可按需要使用 **播放**、**外部播放器** 或 **下载**。

## 播放

IwaraTV 支持两种播放模式：

- **MPV**：默认推荐模式。应用会依次查找设置中指定的 MPV、Windows 打包内置 MPV、开发环境的 `vendor/mpv/mpv.exe`，以及 `PATH` 中的 MPV。
- **外部播放器**：配置播放器可执行文件路径和启动参数模板。模板必须包含 `{url}`，否则播放器收不到视频直链。

常用外部播放器占位符：

- `{url}`：视频媒体直链。
- `{title}`：视频标题。
- `{headers}`：带 Iwara referrer 的 HTTP 头字符串。

如果选择的清晰度不可用，IwaraTV 会自动回退到最佳可用格式，并提示发生了回退。

## 登录与验证

部分操作需要 Iwara 账号或浏览器会话：

- 查看订阅作者视频。
- 关注或取消关注作者。
- 发送评论或回复。
- 访问需要 Iwara cookie 或站点 token 的视频。

打开 **设置**，使用 Iwara 验证操作。在弹出的应用内窗口完成登录或 Cloudflare 验证。IwaraTV 捕获到可用站点 token 后，窗口会自动关闭，后续请求会复用 token 和 cookie。

凭据说明：

- API 登录 token 会在可用时保存到操作系统钥匙串。
- 网页会话数据通过应用 WebView 会话处理。
- 如果安全存储不可用，应用会提示登录持久化可能受限。

## 搜索、标签与列表

浏览列表可以结合关键词和标签过滤。**设置** 中的标签偏好会影响列表视图：

- **关注标签** 用于关注标签列表。
- **屏蔽标签** 会隐藏命中的视频，并且优先级高于关注标签。
- **扫描页数** 控制多标签或屏蔽标签过滤时扫描多少页列表。
- **请求间隔** 会在扫描多页时加入暂停，减少请求突增。

搜索结果基于 Iwara 搜索，并叠加本地标签和屏蔽标签过滤。

## 评论与作者

视频详情页可以加载评论和回复。完成登录或会话验证后，可以发送一级评论或回复。

作者相关操作会出现在视频详情页和作者列表中。关注或取消关注作者需要登录或会话验证。

## 收藏

使用视频卡片或详情页中的星标/收藏操作，可以将视频保存到本地。**收藏** 视图会集中展示已保存的视频，方便快速重新打开、播放或下载。

收藏是本地数据，不是 Iwara 服务端收藏列表。收藏视图包含：

- **备份**：在应用本地收藏数据旁写出一个备份 JSON 文件。
- **导出**：选择一个 JSON 保存位置，用于共享或手动备份。
- **导入**：将 JSON 收藏文件合并到当前本地列表，并按视频 ID 去重。

导入时会尽量保留现有本地备注，并用导入文件中的内容补全缺失元数据。

## 下载

在 **设置** 中配置下载目录和默认清晰度。详情页下载会使用当前选择的清晰度；视频卡片下载会使用默认清晰度。

下载管理支持：

- 活跃任务状态和进度。
- 首选清晰度不可用时自动回退。
- 便于恢复的临时分片/部分文件。
- 文件足够大且服务端支持 Range 时进行分段下载。
- 对失败历史记录重试。
- 打开已下载文件或所在文件夹。
- 删除下载历史，并可选择同时删除本地文件。

下载历史会保存在应用数据目录中。

## 媒体线路与诊断

Iwara 媒体 URL 可能解析到不同媒体域名。你可以在 **设置** 中：

- 维护候选媒体域名池。
- 用当前视频作为样本执行测速。
- 在还没有线路排序时启用自动测速。
- 播放时将链接替换为测速最快的域名。

如果开启线路替换后播放失败，请关闭替换并使用原始媒体链接重试。

诊断工具可帮助排查清晰度和会话问题：

- **X-Version 嗅探**：刷新 Iwara API 请求所需的盐值。
- **视频诊断**：检查可用清晰度，并在可能时从嵌入页面抓取 Iwara 网络响应。
- **媒体测速报告**：展示每个域名的成功状态、延迟和吞吐。

## 本地数据

IwaraTV 会在应用数据目录写入本地状态：

- `settings.json`：播放器、下载、Iwara、媒体线路、标签和历史设置。
- `downloads.json`：下载历史。
- `favorites.json`：本地收藏。
- 钥匙串条目：系统钥匙串可用时保存的 API token。

下载的视频会保存到配置的下载文件夹；默认情况下会使用系统下载目录下的 `IwaraTV` 文件夹。

## 常见问题

- **提示 "Tauri API is not available"**：通常是运行了 Vite 网页预览。请使用打包桌面应用，开发时使用 `npm run dev:tauri`。
- **找不到 MPV**：在 **设置** 中选择 `mpv.exe` 或 `mpv`，将 MPV 安装到 `PATH`，或改用外部播放器。
- **外部播放器无法打开**：确认可执行文件存在，并且启动参数模板包含 `{url}`。
- **订阅列表或评论失败**：完成应用内 Iwara 验证流程，然后刷新应用状态。
- **只显示低清晰度格式**：登录后使用视频诊断；部分会话格式可能需要捕获浏览器凭据。
- **下载失败或卡住**：从下载历史重试，降低最大连接数，关闭线路替换，或选择其他清晰度。
- **线路替换后出现 403**：关闭媒体域名替换，使用原始链接。
- **导入收藏时跳过条目**：确认 JSON 来自 IwaraTV 导出，或至少包含有效视频 ID。
